# react-native-emoji-container

Simple Emoji Picker for React Native Android and IOS

## Installation

```sh
npm install react-native-emoji-container
```

## Usage

```js
import EmojiContainer from 'react-native-emoji-container';

// ...

<EmojiContainer />
```
